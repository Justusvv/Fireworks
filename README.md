# Fireworks

## Description
This project is a multicolor visualization of fireworks as a screensaver in Python. It uses the `pygame` library to create a graphical display of fireworks that explode in various colors and patterns.

## Instructions
1. Ensure you have all the dependencies installed (see below).
2. Run the `fireworks.py` script to start the screensaver.

## Dependencies
- `pygame`
- `random`
